# Projeto DIO NotebookLM
Primeiro projeto do bootcamp Riachuelo - Cibersegurança


## Objetivo do projeto
O projeto propõe a criação de um caderno temático no NotebookLM, reunindo de três a cinco fontes sobre um assunto financeiro, do qual nesse projeto vai ser de Educação Financeira.

## O que é o NotebookLM
Um NotebookLM é uma ferrameta do Google de Inteligência Artificial que pode ser usada para auxiliar nos estudos. 
Nele podemos criar cadernos, podcasts ou até mapas mentais.
<img width="1343" height="628" alt="imagem notebookLM01" src="https://github.com/user-attachments/assets/d45a8cf3-580d-4c4b-9460-c711cc75eba9" />

O NotebookLM usa as fontes que a gente indica a ele. Assim as resposta que ele fornece, vem das fontes indicadas por nós. 
Podemos enviar fontes através de documentos em vários formatos, como PDF por exemplo. Videos do youtube servem de fonte também, além de sites na internet.
O NotebookLM tem uma função de descobrir fontes. Com essa função ele pode fazer uma busca e sugerir opções de fontes.
